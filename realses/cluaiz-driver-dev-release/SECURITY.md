# 🛡️ Security Policy

Cluaiz is built for sovereign integrity. We take security seriously and appreciate any help in identifying vulnerabilities.

## 🏛️ Supported Versions
We currently support and provide security updates for the latest version of the Cluaiz core only.

## 🛑 Reporting a Vulnerability
**DO NOT open a public issue for security vulnerabilities.**

If you discover a security-related bug or a way to bypass the **CSL Architectural Lock**, please report it privately:

1.  **Email**: Send a detailed report to **hello@cluaiz.com**.
2.  **Disclosure**: Give us a reasonable amount of time (90 days) to address the issue before any public disclosure.
3.  **Recognition**: Valid security researchers will be acknowledged in our Hall of Fame and may be considered for the **Contributor Commercial Waiver**.

## 🚀 Our Commitment
We will acknowledge your report within 48 hours and provide a timeline for the fix. Our goal is to ensure the **Sovereign Shield** of Cluaiz remains unbreakable.
