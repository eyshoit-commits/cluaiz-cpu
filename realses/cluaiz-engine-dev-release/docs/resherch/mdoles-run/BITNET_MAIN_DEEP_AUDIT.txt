### 2. ⚡ KERNEL LEVEL TRACE (THE TERNARY ADDER)
Traditional GPU kernels use `FMA` (Fused Multiply-Add). BitNet uses `Sovereign ADD-ONLY` math.

#### 2.1 The CUDA Ternary Kernel (Draft Specification)
```cuda
extern "C" __global__ void bitnet_ternary_gemm(
    const int8_t* __restrict__ weights, // Packed ternary
    const half* __restrict__ inputs,
    half* __restrict__ output,
    int N, int K
) {
    int row = blockIdx.y * blockDim.y + threadIdx.y;
    int col = blockIdx.x * blockDim.x + threadIdx.x;
    
    float acc = 0.0f;
    for (int k = 0; k < K / 4; ++k) {
        int8_t w8 = weights[row * (K/4) + k];
        // Unpack 4 ternary weights
        for (int i = 0; i < 4; ++i) {
            int8_t w = (w8 >> (i * 2)) & 0x03;
            float weight = (w == 1) ? 1.0f : (w == 2) ? -1.0f : 0.0f;
            acc += weight * (float)inputs[k * 4 + i];
        }
    }
    output[row * N + col] = (half)acc;
}
```
